## 🔧 Quick Reference

### Essential Commands
```bash
# Basic usage
./server-stats.sh

# Install system-wide
sudo ./install.sh

# Test installation
./test-server-stats.sh

# View help
./server-stats.sh --help
```

### Key Files
- `server-stats.sh` - Main monitoring script
- `install.sh` - Installation script
- `server-stats.conf.example` - Configuration template
- `README.md` - Project overview
